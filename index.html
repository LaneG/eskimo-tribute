Repeating SVG Background

This website features a repeating SVG background created with Gerstnerizer, a free online tool from Andreas Köberle.

How I did it

Here are some notes about how I made the background for this page:

  1. I had best results with the square (middle) pattern. I was not able to get the triangle or hexagon pattern to repeat properly in my browser.
  2. I used grid settings 6,6,100. That was enough instances I could see how it would repeat, and it fit on my screen.
  3. To draw lines, click and drag on the grid in the left column. To remove lines alt+click on the lines.
  4. Be careful not to click the "back" button or reload the page or you will lose all your work.
  5. When ready to export, change the grid to 1,1,100 and save using "command+shift+s" to save the file as .svg.
  6. Open the .svg file in Adobe Illustrator. Edit the artboard to be the size of the image and adjust the line colors as you like. I am using two shades of blue for a subtle effect.
  7. I added the background to my css file as: 
    8. 
          body { 
          background-image: url('pattern.svg');
          background-repeat: repeat-y repeat-x;
          }

A few more tips

The tool is fun to use, but a little hard to figure out how it works. I noticed it worked best in Google's Chrome browser. I suggest you click "Random Lines" a few times to get the hang of it. I wasn't able to get the "Search for palettes feature working. I expected that I'd be able to change the color of my lines, or maybe even color different parts of the image. It didn't work as expected. Typing a word in the box brings up a drop down of palettes, but clicking on anthing in the drop down just dismisses the palettes with now change in line color.
